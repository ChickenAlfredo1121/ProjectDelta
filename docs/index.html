How to Make Canvas' Calendar Better

Canvas is an app/website used by Universities as a class management and access system there are many good functions to it primarily its side bars information. This includes the smaller calendar, tabs you see on calendar, and other calendars. There is no apparent issues and the design is good with these so they will be exempt from the changes. Additionally, the "edit Event" is well formated with simple and ample options so for the same reasons it is left out. Where the main issues are is in the calendar itself. When you click on an assignment it shows a small box that you have to use horizontal and vertical sliders to see, it also redirects the current page to the assignment. Additonally, if you click off the assignment on the calendar itself it will automatically open up the "edit Event". The metric for how the updated one below the Current is time. Having to reopen the calendar every time to see it instead of the page redirecting itself is tedious and can be avoided. While I would love to simulate this i don't have a comparison to the original, but the ease shows for itself in the interaction you will face yourself in the updated version.
